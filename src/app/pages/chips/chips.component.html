ngym-chips
Chips library built with Angular 13. Download from here.
Avaliable Properties
# Name Type Description
1 [(ngModel)] Tag[] last value is set to a Tag[]
2 [placeholder] string Any string ' ' can be used. Ex: 'Add Recepient'
3 [duplicateAllowed] boolean Duplicate chips or not
4 [dragZone] string Should be unique for dragging. Ex: 'zone1' and 'zone2'
5 [splitChars] string[] [',', ';'] these are supported at the moment
6 [dragAllowed] boolean dragging is optional. false or true
7 [appendToBody] boolean setting true can correct css issues
8 [pattern] RegExp Ex: number regex for chip validation. /^\d*$/.test(value)
9 (onAddEmitter) event event after adding a chip.
10 (onKeyPressedEmitter) event Ex: changeAutoCompleteItems($event)
11 name string should be unique
